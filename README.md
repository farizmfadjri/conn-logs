# conn-logs
Connection logs for CSGO Server

## Install Guide
- Copy `include`,`discord`,`dcLogger.sp` To `csgo/addons/sourcemod/scripting/`
- Copy `dcLogger.smx` To `csgo/addons/sourcemod/plugins/`
